## 2. Projekta Mērķis
Izstrādāt modernu, drošu un ērtu sistēmu sociālās aprūpes iestādēm, kas automatizē dokumentāciju un atvieglo darbinieku ikdienu.

## 3. Darba Process (Roadmap)

Mūsu darba plāns sadalīts 4 posmos:

1.  **Ideja un Izpēte (Zane):** Definēt problēmu (papīra kalni SAC) un izpētīt tirgu.
2.  **Tehnoloģijas un Loģika (Aleksandrs):** Izvēlēties rīkus (Java, SQL) un aprakstīt, kā sistēma strādās.
3.  **Prototipēšana (Sabīne):** Izveidot vizuālo prototipu (izmantojot gatavās programmas skatus) un lietotāja stāstus.
4.  **Prezentācija un Nodošana (Visi):** Apkopot visu dokumentācijā un sagatavoties aizstāvēšanai.
